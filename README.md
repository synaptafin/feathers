## Development Server

Start the development server on [http://localhost:3000](http://localhost:3000)

Running Feathers alone without templates won't be really useful.   

```bash
# npm
npm run dev

# pnpm
pnpm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run generate

# pnpm
pnpm run generate
```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview
```
